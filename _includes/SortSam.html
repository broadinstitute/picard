SortSam

Sorts a SAM or BAM file. This tool sorts the input SAM or BAM file by coordinate, queryname or some other property of the SAMRecord. Input and output formats (SAM or BAM) are determined by the file extension.

Usage example:

java -jar picard.jar SortSam \
      I=input.bam \
      O=sorted.bam \
      SORT_ORDER=coordinate

OptionDescription
INPUT (File)The BAM or SAM file to sort. Required.
OUTPUT (File)The sorted BAM or SAM output file. Required.
SORT_ORDER (SortOrder)Sort order of output file Required. Possible values: {unsorted, queryname, coordinate, duplicate}
