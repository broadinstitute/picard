Overview of Picard command-line tools

The Picard command-line tools are packaged as a single executable jar file. As of version 2.0.1 (Nov. 2015) Picard requires Java 1.8 (jdk8u66). The last version to support Java 1.7 was release 1.141. They can be invoked as follows:

java jvm-args -jar picard.jar PicardCommandName OPTION1=value1 OPTION2=value2...

Most of the commands are designed to run in 2GB of JVM, so the JVM argument -Xmx2g is recommended.
