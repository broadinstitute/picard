Tools that use Picard

picardmetrics

picardmetrics is a Bash script that simplifies calling Picard tools on a BAM file:

  • Generates the input files required by CollectRnaSeqMetrics.
  • Run up to 12 Picard tools on a BAM file.
  • Collate the many different output files generated by Picard.
